## CHOCOLATE SALES DATA ANALYSIS ( INTERACTIVE DASHBOARD USING PowerBI)
## PROJECT OBJECTIVE
This project aims to develop an interactive Power BI dashboard to analyze chocolate sales data effectively.It provides insights into sales trends across time, products, countries, and sales personnel.The dashboard highlights key performance metrics such as total sales and units sold.
It supports data-driven decision-making through clear and structured visualizations.

## KPIS
- What is the total sales amount and total number of boxes sold during the selected period?
- How are sales trending over time by year, quarter, and month?
- Which chocolate products contribute the highest and lowest sales revenue?
- Which countries generate the highest share of total sales?
- How does individual salesperson performance compare in terms of sales amount?

## Process
-	Collected and imported raw chocolate sales data into Power BI.
-	Cleaned and transformed data using Power Query for accuracy and consistency.
-	Created calculated measures and KPIs to analyze sales performance.
-	Designed interactive visualizations to track trends by time, product, country, and salesperson.
-	Validated insights and published the dashboard for reporting and analysis.
